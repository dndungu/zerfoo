## Architecture Notes:
- Graph builders live in inference/arch_{llama,gemma,common}.go
- buildTransformerGraph (arch_common.go) is the shared loop for all architectures
- Graph accepts token IDs [1, seqLen] and does embedding lookup internally via embeddingLookupNode
- Gemma applies sqrt(hidden_size) scaling in the embedding lookup node
- LoadFile() is the entry point for GGUF files: parse -> extract tokenizer -> build graph -> wire Generator
- buildArchGraph dispatches to architecture-specific builder based on GGUF general.architecture
- K-quant storage types: Q4KStorage (144B/256val), Q5KStorage (176B/256val), Q6KStorage (210B/256val)
- All K-quant formats in tensor/quantized_kquant.go, wired in model/gguf/loader.go
